# 🧠 Sleep EEG latent representations for apnea-hypopnea index estimation in ischemic stroke patients

🌟 Overview

This is a Research Internship under Prof. Ayush Tripathi, IIT Indore, investigating whether overnight sleep EEG signals can predict brain health and cognitive outcomes.

Sleep EEG contains rich neurophysiological signals — spindle activity, delta power, slow oscillations — that encode cognitive state, disease risk, and brain health. This project extracts that information using the Philosopher's Stone EEG Foundation Model and evaluates its predictive power using rigorous ML pipelines across 100 subjects.

**The data associated with this repository can be accessed via:**
https://www.nature.com/articles/s41597-026-06747-w
